# Tech Stack
A **tech stack** (or technology stack) is the complete set of technologies, tools, programming languages, frameworks, and platforms used to build and run an application, website, or software product. [^1]
Saat penulisan [[1763279341-product-requirements-document-prd|PRD]], pengambilan keputusan untuk Tech Stack ini didasari oleh pengalaman. *I am mainly used PHP or [[1763224944-laravel|Laravel]] as a framework but i know like any ecosystem in other language like in NodeJS with BullMQ for queueing things* . Tapi penggunaan Tech Stack tadi menyesuaikan bagaimana *scope* aplikasi tersebut. Misalkan, saya tidak akan menggunakan PHP atau di sini [[1763224944-laravel|Laravel]] jika aplikasi tersebut hanya bertugas untuk *web automation* saja. Ya, memang ada cara atau dengan PHP bisa kita lakukan untuk *web automation* ini, tapi kendalanya ada di pertanyaan berikut "*You sure use Laravel for just tiny things?*". Kecuali jika kita ingin membangun *project* *web automation* tadi yang dapat di kelola atau memiliki banyak fitur lainnya.
