# Openwrt
Openwrt berdasarkan hasil observasi saya adalah suatu interface untuk pengelolaan jaringan. Seperti jika menggunakan router Indihome yang basisnya adalah Huawei maka tampilan atau interfacenya adalah sesuai dengan Huawei punya atau Huawei buat. Openwrt ini berfungsi untuk me-*replace* OS router tertentu sehingga dibanding OS kita menggunakan atau interface menggunakan milik Huawei, kita menggunakan Openwrt.
