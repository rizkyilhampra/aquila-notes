# Database connection strings examples
## MySQL
### User and password
```
mysql://user:pass@host:port/dbname
```
### Socket
```
mysql:/var/run/mysqld/mysqld.sock
```
## SQLite
```
/path/to/sqlite/file/dbname.db
```
